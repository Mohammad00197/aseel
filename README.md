# 🤍

موقع بسيط بأربع أسئلة… الجواب دايمًا "نعم" — وزر "لأ" بيهرب 😄

A tiny mobile-first Arabic Q&A page. Every question only accepts *yes*; the *no* button playfully dodges left and right. Built as a single self-contained `index.html`.

## تشغيل / Run
Just open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## نشر / Deploy (GitHub Pages)
Settings → Pages → Branch: `main` → `/ (root)` → Save. الرابط بيصير جاهز خلال دقيقة.
